# Honeyvore
Ever looked at a perfectly good Lox Meat Pie and thought "but where's the honey?" Well, now you can live the dream. This mod ensures that your Viking only consumes food and potions that contain honey somewhere in their crafting chain. Because "If it doesn't have honey, it's not worth eating" - Gwen the Shieldmaiden.

## Installation (manual)
Drop the dll into your BepInEx\plugin folder and embrace the sweet, sweet lifestyle. Beekeeper suit not included.

## Features
The mod will block you from consuming anything that has no honey in it's production chain. Or is actually pure honey. That's it. Will show subjectively semi funny messages while doing so.
